# How to use?
- [Download & Install Postman](https://www.postman.com)
- Download the Drupal-7/8/9 postman collection
  - Click on the above collection link
  - Click on 'RAW' button
  - Save the page as JSON file
- Open Postman
- Click 'Import' Button
- Click 'Choose Files'
- Select saved JSON file
- You are Done!
